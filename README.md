# 🌕emoji-prefix
a obsidian plugin with a simple funciton: **add or remove emoji🌕 prefix for mardown header**

## usage

shortcut:
- add prefixes: `Ctrl + shift + U`
- remove prefixes: `Ctrl + shift + R`
- defautlt prefiex(Sort by title level incrementally):
  1. "🌕",
  1. "🌖",
  1. "🌗",
  1. "🌘",
  1. "🌑",
  1. "⭐"
